# To-Do App

A simple and interactive **To-Do App** built with **HTML, CSS, and JavaScript**.  
It allows users to **add, delete, and mark tasks as complete**, with tasks persisting in the browser using `localStorage`.

---

## 📝 Features

- Add new tasks to the list
- Delete tasks individually
- Mark tasks as completed
- Persist tasks in localStorage
- Clean and responsive UI

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage for data persistence

---

## 🚀 Live Demo

You can open `index.html` in your browser to try it locally, or deploy it on **GitHub Pages**.
